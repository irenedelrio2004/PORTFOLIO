## Respuesta de la IA (plan)

1. Definir la estructura de carpetas del proyecto (css, js, assets, docs).
2. Crear un archivo `index.html` con la estructura básica HTML5.
3. Incluir Bootstrap mediante CDN.
4. Usar el sistema de grid para organizar el layout principal.
5. Verificar que el proyecto sea responsive desde el inicio.
