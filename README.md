## personalización/ editar:

1. Contenido
   Edita los archivos HTML (index.html, about.html, projects.html, contact.html) para cambiar los textos.
   Sustituye los títulos, párrafos y enlaces por tu propia información.
   Añade o elimina secciones según tus necesidades.
   Cambia las imagenes por tuyas personales y el nombre principal del menu por tu nombre.

2. Estilos
   Los estilos están organizados por secciones dentro de assets/css/.
   index.css contiene los estilos generales.
   Cada página tiene su propio archivo CSS (about.css, projects.css, contact.css).
   Puedes cambiar colores, tipografías y espaciados desde los archivos CSS.

3. JavaScript
   El archivo assets/js/main.js controla las interacciones básicas (menú, navegación, etc.).
   Puedes añadir nuevas funcionalidades.

4. Proyectos
   Para añadir un nuevo proyecto, edita projects.html.
   Duplica una tarjeta existente y cambia el contenido (imagen, título, descripción y enlaces).
   Asegúrate de mantener la misma estructura HTML para conservar el diseño.

5. Página 404
   El archivo 404.html muestra una página de error personalizada.
   Puedes modificar el texto o el diseño.
